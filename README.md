# The AI-First Company Blueprint

An operating model for running a company where AI agents are first-class participants, not tools.

**Author:** [Gilad Pagi](https://www.linkedin.com/in/giladpagi/)
**Version:** 3
**Date:** March 2026
**Website:** [BeanOS.ai](https://beanos.ai/blueprint/)

---

## What This Is

This document describes an operating model for a company where AI agents do the work and humans make decisions. It is not about using AI to assist employees — it is about building the company so that agents operate it, and humans steer.

The document is organized in two parts:
- **Part I** establishes foundational principles — tool-agnostic and durable.
- **Part II** presents one opinionated implementation with specific technology choices.
